# CANGuard-AI

CANGuard-AI is a CAN bus intrusion detection system with a web dashboard.

## Project structure

- `arduino/` - Arduino integration and CAN bus hardware code.
- `ids/` - IDS application modules.
- `dashboard/` - Flask dashboard web app.
- `database/` - SQLite database and schema.
- `logs/` - Application logs.
- `docs/` - Documentation.
